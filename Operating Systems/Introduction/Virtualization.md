-> It is used by Operating system , to achieve something which is somehow related to infinite supply of whether it can be RAM , CPU etc.

Example:
1. Virtualization of CPU: Here , Multiple programs can be run at the same time , let's say in a single core CPU. On the other hand , it is observed that there is only one CPU , then how it is running multiple programs at the same time. It is achieved by virtualization of CPU.
2. Virtualization of Memory: Here , multiple program has it's own private address space. As, the memory is a shared resource. Then , it will be interesting to know , that how the OS actually achieve this part of virtualizing the memory.
